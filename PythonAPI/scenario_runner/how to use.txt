---- Installation (Windows 10) ----

Make new enviroment variablle called PYTHONPATH and add paths:

.. meaning your carla path!

..carla\PythonAPI\carla
..carla\PythonAPI\carla\dist
..carla\PythonAPI\carla\agents
..carla\PythonAPI\carla\dist\<..>.egg


---- How to use (from single script) ---- 

1) Open CMD and locate to ../scenario_runner folder

python wintersim_scenario_runner.py --openscenario /srunner/examples/Slalom.xosc --output

Note. 
While scenario runner loads correct map by itself, it's recommended to open correct scene yourself.
Remember to close scenario runner before stopping play mode!


-- How to use (wintersim control):

1) Open command line and locate to ../scenario_runner folder

python scenario_runner.py --openscenario /srunner/examples/Slalom.xosc --output

python scenario_runner.py --openscenario /srunner/wintersim/break_test.xosc --output

2) Open secod CMD, locate to ..PythonAPI/wintersim_examples folder and run wintersim_control.py with following argument

python manual_control.py --scenario

---- How to use (Carla manual control) ---- 

1) Open command line and locate to ../scenario_runner folder

python scenario_runner.py --openscenario /srunner/examples/Slalom.xosc --output

2) Open secod CMD, locate to ../scenario_runner folder and run manual_control.py

python manual_control.py
